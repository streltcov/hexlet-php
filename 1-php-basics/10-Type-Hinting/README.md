
### src/Number.php

Реализуйте функцию reverse, которая переворачивает цифры в переданном числе:

    <?php
    
    use function Number\reverse;
    
    reverse(13); // 31
    reverse(-123); // -321

Не забудьте задать тип входного и выходного аргумента.
